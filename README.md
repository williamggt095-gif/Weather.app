# 🌦️ Weather App

> Aplicação web que exibe informações climáticas em tempo real usando API.

---

## 🚀 Funcionalidades

* 🔍 Busca por cidade
* 📍 Localização automática (GPS)
* 🌡️ Temperatura em tempo real
* ☁️ Descrição do clima
* 🎨 Ícones dinâmicos
* 🌙 Modo escuro automático
* 💾 Salvamento da última cidade
* 📱 Layout responsivo
